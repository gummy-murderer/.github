## 🚀 프로젝트 소개
살곰사건이 벌어진 베어머더러 마을! 저명한 탐정(플레이어)에게 의뢰를 요청하여 살곰마를 잡으려 합니다.
탐정은 베어머더러 마을에 도착해서 시체를 조사하고 주민을 심문하며 범인을 찾아냅니다.
마을을 뒤집고 다니는 탐정이 마음에 들지 않았는지 범인은 마을 사람들을 한곰씩 살해하기 시작합니다.
과연 탐정은 범인을 찾아 마을 주민을 지켜낼 수 있을까요?
